# Proyecto ConnectaTel: Análisis de Clientes y Patrones de Uso

## Objetivo del Proyecto

El objetivo de este proyecto es analizar el comportamiento de los clientes de ConnectaTel a partir de información demográfica y registros de uso de servicios de telecomunicaciones.

Para ello, se realiza un proceso de limpieza, transformación y análisis exploratorio de datos con el fin de:

- Identificar patrones de consumo.
- Segmentar usuarios según su nivel de uso y edad.
- Detectar valores atípicos.
- Generar insights para la toma de decisiones del negocio.

## Herramientas Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Principales Resultados

- Limpieza y estandarización de variables demográficas y de uso.
- Identificación de segmentos por edad y nivel de consumo.
- Detección y análisis de outliers relevantes para el negocio.
- Generación de insights y recomendaciones basadas en datos.

## Estructura del Proyecto

- `S7_Version_Estudiante_Project_ConnectaTel.ipynb`: notebook principal con el análisis completo.
- `README.md`: descripción general del proyecto.

## Conclusiones

El análisis permitió identificar distintos perfiles de clientes y patrones de consumo que pueden utilizarse para diseñar planes más adecuados y estrategias de retención más efectivas.
