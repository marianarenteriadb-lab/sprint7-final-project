Proyecto ConnectaTel: Análisis de Clientes y Patrones de Uso
Objetivo del Proyecto

El objetivo de este proyecto es analizar el comportamiento de los clientes de ConnectaTel a partir de información demográfica y registros de uso de servicios de telecomunicaciones. Para ello, se realiza un proceso de limpieza, transformación y análisis exploratorio de datos con el fin de identificar patrones de consumo, segmentar usuarios según su nivel de uso y edad, detectar valores atípicos y generar insights que apoyen la toma de decisiones del negocio.

A través de este análisis se busca:

Evaluar la calidad de los datos e identificar problemas que puedan afectar los resultados.
Analizar el comportamiento de los usuarios mediante métricas de llamadas, mensajes y minutos consumidos.
Identificar segmentos de clientes con características similares.
Detectar usuarios con patrones de consumo atípicos.
Generar recomendaciones para optimizar la oferta de planes y mejorar las estrategias de retención y crecimiento de clientes.
Herramientas Utilizadas
Python
Pandas
NumPy
Matplotlib
Seaborn
Principales Resultados
Limpieza y estandarización de variables demográficas y de uso.
Identificación de segmentos por edad y nivel de consumo.
Detección y análisis de outliers relevantes para el negocio.
Generación de insights y recomendaciones para ConnectaTel basadas en evidencia de datos.
